== Description ==
Das etablierte Tool für Ihren digitalen Immobilieneinkauf. Im Austausch gegen seine Kontaktdaten erhält der Eigentümer von Ihnen eine Wertermittlug seiner Immobilie. In der Pro Version erhält der Eigentümer in Echtzeit eine Vergleichswertermittlung inkl. einer Wohnmarktanalyse. Sie erhalten im Gegenzug die Kontaktdaten per E-Mail. Einmal in Ihre Website eingebaut, funktioniert der Leadgenerator voll automatisiert.


== Installation ==
Lade den kompletten prt Ordner in dein /wp-content/plugins/ Verzeichnis.
Aktiviere das Plugin im ‚Plugins‘ Menü in WordPress.
Du findest ein „Property Rating“ Menü im WordPress Adminbereich
